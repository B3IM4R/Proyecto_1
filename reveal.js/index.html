Archivos Planos

Son una colección de información que está almacenada y es accedida de forma organizada en una base de datos.

Características Clave

  • Asequible
  • Flexibilidad
  • Uso Sencillo
  • Búsquedas Rápidas
  • Facilidad de Exportar
    e Importar Datos

EJEMPLOS

Existen muchos tipos, cada uno con su propia estructura para representar información.

Comma Separated Values

Se caracterizan porque están separados por comas y forman una tabla compuesta de filas y columnas.

   

Archivos XML

(Extensible Markup Language)

Permiten definir y almacenar datos de forma compartible, admitiendo el intercambio de información entre sitios web, bases de datos y aplicaciones de terceros.

Beneficios

  1. Respaldo para las transacciones
    interempresariales
  2. Conservación de la integridad
    de los datos
  3. Mejora de la eficiencia de
    búsqueda
  4. Diseño de aplicaciones flexibles

¿Cómo Abrir Archivos XML?

Aunque existen lectores de este tipo de archivo, como XML Explorer, se puede hacer utilizando:

Abre en tu Computadora

(Excel - Bloc de Notas - Navegador Web)
           

Archivos JSON

(JavaScript Object Notation)

Se utilizan para estructurar datos en forma de texto y permiten el intercambio de información entre aplicaciones de manera sencilla, liviana y rápida.

Ventajas al Utilizarlos

  1. Parsing más fácil
  2. Lectura más Sencilla
  3. Archivos más livianos
  4. Más agilidad en la ejecución
    y el transporte de datos

Lenguaje Independiente

Permite que sean utilizados en cualquier tipo de lenguaje de programación, además de poder manipularse en diferentes plataformas.

Archivos Parquet

Es un formato de almacenamiento en columnas y de código abierto, que está enfocado en optimizar el procesamiento y el modelado de los datos.

Propiedades

  • Metadatos
  • Estructuras Anidadas
  • Reduce gran espacio
    del almacenamiento
  • Las columnas se pueden
    leer independientemente

¿Por qué se crearon?

Para admitir la compresión y codificación de manera eficiente, además de estar optimizado para trabajar con estructuras de datos complejas de forma masiva.
